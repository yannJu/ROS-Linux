- 쉘에서 변수를 입력하거나 출력하기 (echo, read .. )
  - 코드에 있는 변수를 사용하려면 `${"(변수명)"}` 나 `$(변수명)`과 같이 작성
- 조건문 구현해보기
  - 조건문 작성시 주의 : `if [(띄어쓰기) ~조건 (띄어쓰기)]`
- 반복문 구현해보기
  - 반복문에서 문자열을 리스트로 변환해보기
